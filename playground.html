The Code Lab

Real Python, running entirely in your browser. Write something, press Run (or Ctrl+Enter), and watch it execute. Nothing is uploaded — it's just you and the interpreter.

📖 How to use it
Load an example:
🐍 main.py
▶ Output Engine idle
Output will appear here. Press Run to start — the first run downloads the Python engine once (a few seconds), then it's instant.

🔒 Your code runs locally in your browser via Pyodide (WebAssembly). It is never sent to any server.
