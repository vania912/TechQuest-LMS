Variables & types

A variable is a named box that stores a value. Python figures out the type for you. The most common types are text (str), whole numbers (int), decimals (float) and true/false (bool).

🐍 variables.py
Note: You never declare the type yourself — assign a value and Python infers it. This is called dynamic typing.

Working with text

You can join, repeat, and format strings. f-strings (the f"..." form) let you drop variables straight inside text using curly braces.
